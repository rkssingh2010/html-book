HTML

DOCTYPE Decleration

<!DOCTYPE html>: Specifies the version of HTML being used.

Document Meta:

<Meta>: Provides metadata about the HTML document.

Responsive Design:

Using meta viewport tag for responsive design: <meta name="viewport" content="width=device-width, initial-scale=1.0".>

Document Structure:

<html>:The root element of an HTML document.
<head>:Contains meta-information about the HTML document.
<title>:Sets the title of the HTML document.
<body>:Contains the content of the HTML document.

Text Elements:

<h1> to <h6>:Headings, where <h1> is the largest and <h6> is the smallest.
<p>:Paragraphs.
<a>:Hyperlinks.
<strong> and <em>:Used for emphasizing text.

Lists:

<ul>:Unordered list.
<ol>:Ordered list.
<li>:List item.

Images:

<img>:Embeds images in the document. It is a self closing tag eg: <img href=”” alt=””>

Forms:

<form>:Wraps a set of form elements.
<input>:Allows user input. Various types include text, password, checkbox, radio, email, number etc.
<label>:Labels for form elements.
<textarea>:Multiline text input.
<select>:Dropdown list.
<button>:Represents a clickable button.

Semantic Elements:

<article>:Represents a self-contained piece of content that could be distributed and reused independently.
<section>:Represents a generic section of content.
<header> and <footer>:Defines the header and footer of a section or page.
<nav>:Represents a navigation menu.
<aside>:Represents content that is tangentially related to the content around it.
<main>:Represents a wrapper in the body.
