HTML

DOCTYPE Decleration

<!DOCTYPE html>: Specifies the version of HTML being used.

Document Meta:

<Meta>: Provides metadata about the HTML document.

Responsive Design:

Using meta viewport tag for responsive design: <meta name="viewport" content="width=device-width, initial-scale=1.0".>

Document Structure:

<html>:The root element of an HTML document.
<head>:Contains meta-information about the HTML document.
<title>:Sets the title of the HTML document.
<body>:Contains the content of the HTML document.

Text Elements:

<h1> to <h6>:Headings, where <h1> is the largest and <h6> is the smallest.
<p>:Paragraphs.
<a>:Hyperlinks.
<strong> and <em>:Used for emphasizing text.
